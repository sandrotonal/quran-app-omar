# Frontend

React + Vite frontend for Quran Semantic Graph.

## Setup

```bash
npm install
```

## Environment Variables

Create `.env` file:

```env
VITE_API_URL=http://localhost:3001
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Components

### AyetSelector

Sure ve ayet seçim formu.

### SemanticGraph

React Flow ile interaktif graph görselleştirme.

### AyetDetailPanel

Seçilen ayetin detaylarını gösteren drawer.

## Features

- Dark mode toggle
- Responsive design
- Graph zoom & pan
- Node click for details
- Similarity score visualization
